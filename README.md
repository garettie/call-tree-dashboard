# Call Tree Dashboard

A real-time monitoring tool built with React and Supabase for tracking employee safety during emergency or business disruptions. The application provides a live view of active incidents and an archive for historical data.

## Features

- **Live Dashboard**: Monitors active incidents with real-time updates and clear visual indicators.
- **Incident Management**: Controls for starting and ending incidents directly from the interface.
- **Historical Archive**: A dedicated view for reviewing past events and analyzing historical performance.
- **Data Visualization**: Integrated charts using Recharts for quick assessment of metrics and trends.

## Tech Stack

- **React 19**
- **Vite**
- **TypeScript**
- **Supabase**
- **Tailwind CSS 4**
- **Lucide React**

## Getting Started

### Prerequisites

You will need a Supabase project and the following environment variables configured. Create a .env file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Installation

Install dependencies using npm:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

### Building for Production

Create an optimized build:

```bash
npm run build
```

The output will be generated in the dist directory.
